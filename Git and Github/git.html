Git - Version Control System - Helps you maintain different versions of your code.

GitHub -Website that works with Git

Whenever we run 'git init' it creates a hidden directory .git which can be listed by ls -a which tracks our application

git status - Gives the status of our files in our repo and info on current directory(branch)

git add -Adds to commit list which will be traked (Changes to be committed)

git commit -m"Message" -Commits uncommited files which can be viewed through git status

git add . - Add all files in the directory to the commit list

git log -To get the history logs of our commits

git checkout "commit-id" - To revert back to the older version by pointing the head pointer at it

git checkout master -To go forward in time to master branch where we commit

git revert --no commit "commit id"..HEAD - Revert to past commit state
